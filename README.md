# Unibomb

SMS bombing system from Unicorx by sheikh haziq
## Installation

Use NPM to install Unibomb.

```bash
npm i -g unibomb
```
## or
Use YARN to install Unibomb.
```bash
yarn global add unibomb
```
## Usage

```bash
unibomb start
```
